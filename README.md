# Social Sandbox

A Vite, React, and TypeScript tweet viewer.

## Commands

```bash
npm install
npm run dev
npm run test
npm run build
```

## Project structure

```text
src/
  App.tsx                 Root app with sample tweets and like/retweet state
  main.tsx                React entry point
  components/Tweet.tsx    Tweet display component
  types/tweet.ts          Tweet data types
  test/setup.ts           Vitest test setup
public/avatars/           SVG avatar assets
```
