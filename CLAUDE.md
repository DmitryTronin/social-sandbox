# react-app-1

A React + TypeScript tweet viewer app built with Vite.

## Tech Stack

- React 18
- TypeScript 5
- Vite 5
- Path alias: `@/` → `src/`

## Project Structure

```
src/
  components/    # React components (e.g. Tweet.tsx)
  types/         # TypeScript interfaces (e.g. tweet.ts)
  App.tsx        # Root component
  main.tsx       # Entry point
```

## Commands

```bash
npm run dev      # Start dev server
npm run build    # Type-check + build
npm run preview  # Preview production build
```
